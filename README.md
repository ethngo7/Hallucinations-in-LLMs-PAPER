# Hallucinations-in-LLMs-PAPER

This is a paper I crafted for an NLP class at UC Berkeley. The topic is **hallucinations in LLMs**, covering definitions, history, mitigation strategies, and more. Below is the **Abstract** and a link to the full paper.

**Abstract**

*"This survey examines the problem of hallucination in large language models (LLMs), where hallucination is a model's tendency to generate fluent, plausible text that is factually incorrect or unsupported by any real sources. Hallucination has existed since early sequence-to-sequence architectures to modern transformer-based LLMs. I review how the field has defined hallucination, how it has been evaluated, and the major classes of mitigation strategies that have emerged — covering intrinsic and extrinsic hallucination, faithfulness-oriented evaluation benchmarks, retrieval-augmented generation, training-based methods, and decoding-time interventions."*
